# Rust-WASM Fractal Explorer

A WebAssembly-powered Mandelbrot fractal explorer built with Rust, wasm-bindgen, and the browser Canvas API. Click to zoom in and explore the fractal in real time.

## Features

Fast Mandelbrot rendering in Rust via WebAssembly  
Zoom at mouse click position  
Smooth color gradient  
Fully client-side: no server computation  

## Project Structure

fractal-explorer/
├─ src/lib.rs       # Rust source code (WASM exports)
├─ www/
│  ├─ index.html    # HTML page with canvas
│  └─ index.js      # JS glue: loads WASM, handles clicks
├─ pkg/             # Generated by wasm-pack
└─ Cargo.toml       # Rust dependencies

## Prerequisites

Rust + wasm-pack  
Any static file server to serve www/ (e.g., python3 -m http.server 8000)

## Build

Build the WASM module:

wasm-pack build --target web

Copy the pkg/ folder into www/:

cp -r pkg www/

## Run

Serve the www/ folder:

cd www
python3 -m http.server 8000

Open http://localhost:8000 in your browser.

Click on the fractal to zoom in.
